Static Website Hosting on AWS S3 via GitHub Actions

Project Overview

This website is hosted on Amazon S3 and is automatically deployed using GitHub Actions.

How it works:

  • Step 1: Code is pushed to the GitHub repository.
  • Step 2: GitHub Actions triggers a workflow on every push.
  • Step 3: The AWS CLI syncs the repository files with the S3 bucket.
  • Step 4: The live site is updated instantly!

Deployment Status: Live & Automated
